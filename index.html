Projects

Smart Attendance System

This project involved developing an AI-powered smart attendance system using facial recognition technology. The system is designed to automatically mark attendance by detecting and recognizing students' faces, thereby reducing manual errors and time consumption. It also includes features for generating reports and managing attendance records effectively.

COVID-19 Tracker App

I developed a mobile application that provides real-time updates on COVID-19 cases globally. The app includes features like live statistics, news updates, and a chatbot for answering common questions about the pandemic. The app was built using React Native and integrated with public APIs to fetch the latest data.

E-commerce Platform

This project was focused on building a full-stack e-commerce platform where users can browse, search, and purchase products online. The platform includes user authentication, a secure payment gateway, and an admin panel for managing products and orders. The backend was developed using Node.js and Express, while the frontend was built with React.js.
